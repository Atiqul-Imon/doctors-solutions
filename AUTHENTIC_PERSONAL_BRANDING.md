## 📝 Content Customization Guide

### To Make It Personal:

1. **Add Real Doctor Photo**
   ```html
   Replace the placeholder div with:
   <img 
     src="/images/doctor-photo.jpg" 
     alt="Dr. [Name]" 
     className="w-64 h-64 rounded-full object-cover shadow-2xl border-4 border-white/30"
   />
   ```

2. **Update Doctor's Name**
   - Replace `[আপনার নাম]` with actual name in Bengali
   - Add English version if needed

3. **Update BMDC Registration**
   ```javascript
   bmdc: 'A-12345', // Replace with actual BMDC number
   ```

4. **Update Hospital & Chamber Info**
   ```javascript
   hospital: 'Your Hospital Name',
   address: 'Your Address',
   days: 'Your Working Days',
   hours: 'Your Chamber Hours',
   ```

5. **Update Phone Number**
   - Replace `+880 1XXX-XXXXXX` throughout
   - Update `tel:` link in CTA

6. **Customize About Story**
   - Write 2-3 paragraphs in first person
   - Share treatment philosophy
   - Explain commitment to patients

7. **Update Years of Experience**
   - Change "১৫ বছর" to actual years
   - Update "15+" to match
